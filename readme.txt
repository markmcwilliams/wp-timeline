== Description ==

Simple way to record and display events from the past, the present, and the future!

== Installation ==

1. Upload the `timeline` folder to your `/wp-content/plugins/` directory. Or you can use the in-built plugin search.
2. Activate "Timeline" through the plugins page of your WordPress Admin.
3. Visit "Timeline" from your menu, and create your entries like a normal post.
4. Enjoy ...

== Frequently Asked Questions ==

= What filters do you have available for developers? =

There are currently only two available.

* `timeline_archive` - Alters the `has_archive` argument when we register our Custom Post Type.
* `timeline_rewrite` - Alters the `rewrite` argument when we register our Custom Post Type.

= It doesn't do something, can you add that feature? =

If you think there's something missing, then please do [suggest an enhancement](https://github.com/markmcwilliams/timeline/issues).

It's all happening [over on GitHub](https://github.com/markmcwilliams/timeline/issues).
