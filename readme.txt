== Description ==

Simple way to record and display events from the past, the present, and the future!

== Installation ==

1. Upload the `timeline` folder to your `/wp-content/plugins/` directory.
2. Activate the "Timeline" plugin through the Plugins page of your WordPress Admin.
3. Visit "Timeline" from your menu, and create your events like you would a WordPress Post.
4. Enjoy ...
